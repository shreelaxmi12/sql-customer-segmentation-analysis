# 📊 Customer Segmentation & Sales Analysis (SQL)

## 📌 Project Overview

This project focuses on analyzing customer behavior and sales performance using SQL. The goal is to extract meaningful business insights and segment customers based on their purchasing patterns.

## 🎯 Objectives

* Identify top-performing customers
* Segment customers based on revenue contribution
* Analyze regional and monthly sales trends
* Support data-driven business decisions

## 🛠 Tools Used

* SQL (MySQL)
* Data Analysis Techniques

## 📂 Dataset

The dataset contains retail sales transactions with the following fields:

* Order ID, Order Date
* Customer ID, Customer Name
* Region, State
* Product Category, Sub-Category
* Sales, Profit, Quantity

## 🔍 Key Analysis Performed

### 1. Overall Business Performance

* Total Sales and Profit calculation

### 2. Top Customers Analysis

* Identified top 10 customers based on total sales

### 3. Regional Sales Analysis

* Compared sales performance across regions

### 4. Monthly Sales Trend

* Analyzed sales trends over time

### 5. Customer Segmentation

* Segmented customers into:

  * High Value
  * Medium Value
  * Low Value

## 📈 Insights

* A small group of customers contributes significantly to total revenue
* Certain regions outperform others in sales
* Seasonal trends are visible in monthly sales
* Customer segmentation helps target high-value customers effectively

## 📁 Files in Repository

* dataset.csv → raw dataset
* queries.sql → SQL queries used for analysis

## 🚀 Conclusion

This project demonstrates the use of SQL for real-world business analysis, including customer segmentation and performance tracking.
